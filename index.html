Jbot - A Custom Made Chatbot


In this project, I created a chatbot named Jbot and deployed it online. Jbot randomly answers any question given to it. It was created using chatterbot, a simple chatbot trainer found in Python. The dataset used to train Jbot are English conversations that can be downloaded online. For deployment, flask was used to load the chatbot online. It was hosted using Amazon Web Services and Nginx. This chatbot can be further improved using more advanced pre-trained NLP models like OpenAI's GPT-3 or Google's BERT.

Skills demonstrated: Python, Natural Language Processing, Web Application, Amazon Web Services

Predicting Dota 2 Match Outcomes using ML Classifiers


This project aims to predict the outcome of the game even before the game starts. The models used to predict are Decision Tree (baseline), Random Forest and Gradient Boosting. The baseline shows a 52% accuracy and this study managed to increase this to 60%. The model also showed the top three heroes highly influencing the game (Spectre, Omniknight, Zeus). Frequent Itemset was also used for sanity checking and it showed that the heroes of the winning team are not necessarily the driving heroes. This model can be further improved by using a neural network.

Skills demonstrated: Python, Big Data, ML Algorithms, Frequent Itemset Mining

Implementation of Air Writing Recognition using a 3-Layer CNN


This project will predict the numerical digit being conveyed using hand gesture or air writing.Using the MNIST dataset of 70; 000 handwritten images of the 10 numerical digits, we present a Deep Convolutional Neural Network composed of 3 layers of 2-Dimensional Convolutional Neural Network and 2 layers of Fully Connected Neural Network built using Keras library in Python. Our model was able to achieve an accuracy of 99.74%, precision, recall and F1 scores of 0.99, 0.99 and 0.99 respectively. Furthermore, we achieved a 0.209% improvement in accuracy with the use of augmentation techniques. We further explored the robustness of the model by getting real-time handwritings, captured using OpenCV library. The model was able to get an accuracy of 70%. To further improve the results of the study, it is recommended to increase the training dataset since different people will have different handwriting styles. The development of this system will impact people who have problems communicating because of the environment they are in or because of personal incapacity.

Skills demonstrated: Python, Deep Learning Algorithms, OpenCV

Stock Analysis Trader System


This project is a kind of dashboard for traders. The data was gathered from an API called yfinance where we collected only the top 100 NASDAQ stocks. We also collected news from an RSS feed in Yahoo Finance scraped using python libraries. After collecting these data, we put them all in S3 buckets (4 buckets) to achieve a single source of truth. From the buckets, the data is processed and put in different data warehouses, i.e. RDS, RedShift. From there, the data is transferred to AWS Quicksight. AWS Quicksight is a new product of Amazon for creating a dashboard. The dashboard will be the interface of the users which could help them decide which stocks to buy or to sell.

Skills demonstrated: Python, Data Engineering, Amazon Web Services
